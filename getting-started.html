Getting Started

Connect Your First UPS

  1. Install UPSLink from Google Play.
  2. Configure your NUT server on Synology, Unraid, Proxmox, TrueNAS, Raspberry Pi, or another supported platform.
  3. Open UPSLink and tap Add Connection.
  4. Enter your NUT server IP address, port, and credentials if required.
  5. Tap Discover Devices to find available UPS devices.
  6. Select the UPS device you want to monitor.
  7. Enable notifications for power outages, low battery, runtime warnings, and connection alerts.
  8. Optionally add an Android home screen widget for quick UPS status visibility.
Tip: For Synology users, ensure Enable Network UPS Server is enabled and your Android device IP address has been added to the Permitted DiskStation Devices list.
